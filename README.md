# Vehicle Rental System (Java Swing)

## 📌 Overview

This project is a simple Vehicle Rental System built using Java Swing. It allows users to select a vehicle type, enter the number of rental days, and calculate the total rental cost.

---

## 🚀 Features

* GUI-based application using Java Swing
* Uses Factory Design Pattern
* Supports multiple vehicle types (Car, Bike, Truck)
* Dynamic cost calculation
* Multithreading for processing simulation
* Input validation for rental days

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* Multithreading
* Object-Oriented Programming (OOP)

---

## 📂 Project Structure

* `Project9.java` → Main class with GUI and logic
* `Vehicle` → Interface
* `Car`, `Bike`, `Truck` → Implementation classes
* `VehicleFactory` → Factory class

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Project9.java
   ```
3. Run the program:

   ```bash
   java Project9
   ```

---

## 📊 How It Works

* User selects a vehicle type from the dropdown
* Enters the number of rental days
* Clicks the **Rent** button
* The system calculates total cost based on vehicle rate
* Result is displayed after a short delay
